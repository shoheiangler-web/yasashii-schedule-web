# やさしいスケジュール — セットアップガイド

## 必要なもの
- Mac（iOSビルドにはMacが必須）
- iPhone（実機テスト用、なくてもシミュレーターで可）
- Apple ID（無料で可。App Store申請時は有料アカウント$99/年が必要）

---

## STEP 1: Node.js をインストール

1. https://nodejs.org/ja/ を開く
2. 「LTS版」ボタンをクリックしてダウンロード
3. ダウンロードした `.pkg` ファイルを開いてインストール
4. インストール後、ターミナル（アプリ → ユーティリティ → ターミナル）を開いて確認:
   ```
   node -v
   ```
   `v20.x.x` のような数字が出ればOK

---

## STEP 2: Expo CLI をインストール

ターミナルで以下を実行:
```bash
npm install -g expo-cli eas-cli
```

---

## STEP 3: プロジェクトをセットアップ

1. この `yasashii-schedule` フォルダをデスクトップなど好きな場所に置く
2. ターミナルで以下を実行（フォルダのパスは実際の場所に変更）:
   ```bash
   cd ~/Desktop/yasashii-schedule
   npm install
   ```
   ※ 数分かかります。完了するまで待ってください。

---

## STEP 4: アプリを起動して確認

```bash
npx expo start
```

ターミナルにQRコードが表示されます。

**iPhoneで確認する場合:**
1. App Storeから「Expo Go」アプリをインストール
2. iPhoneのカメラでQRコードを読み取る
3. アプリが起動する

**シミュレーターで確認する場合:**
1. Xcodeをインストール（App Storeから、無料）
2. ターミナルで `i` キーを押す

---

## STEP 5: アプリアイコンを追加

`assets/` フォルダに以下の画像を入れてください:
- `icon.png` — 1024×1024px のアプリアイコン
- `splash.png` — 1284×2778px のスプラッシュ画像
- `notification-icon.png` — 96×96px の通知アイコン
- `ding.mp3` — 完了音（任意）

---

## STEP 6: App Store に申請する前に

### 6-1. Apple Developer Program に登録
- https://developer.apple.com/programs/
- 年額 $99（約15,000円）

### 6-2. app.json を修正
```json
"bundleIdentifier": "com.あなたの名前.yasashiischedule"
```
（例: `com.chochi.yasashiischedule`）

### 6-3. EAS Build でビルド
```bash
eas login
eas build --platform ios
```

### 6-4. App Store Connect で申請
- https://appstoreconnect.apple.com/
- スクリーンショット、説明文、キーワードを入力して提出

---

## ファイル構成

```
yasashii-schedule/
├── App.js                        # ナビゲーション設定
├── app.json                      # Expo設定
├── package.json                  # 依存関係
├── assets/                       # 画像・音声ファイル
└── src/
    ├── screens/
    │   ├── TodayScreen.js        # 今日タブ（メイン画面）
    │   ├── WeekScreen.js         # 週間タブ
    │   ├── AddScreen.js          # 追加タブ
    │   └── SettingsScreen.js     # 設定タブ
    ├── components/
    │   ├── TimeTimer.js          # タイムタイマー（円形）
    │   └── EventCard.js          # 予定カード
    └── utils/
        ├── theme.js              # デザイントークン（色・余白）
        ├── storage.js            # AsyncStorageラッパー
        └── notifications.js     # プッシュ通知
```

---

## 音声認識を本番実装する場合

現在の `AddScreen.js` はUIのみです。実際に動かすには:

```bash
npm install @react-native-voice/voice
```

`AddScreen.js` の `startVoice` 関数のコメントを参照して実装してください。

## トラブルシューティング

**`npm install` でエラーが出る場合**
```bash
npm install --legacy-peer-deps
```

**シミュレーターが起動しない場合**
XcodeのCommand Line Toolsをインストール:
```bash
xcode-select --install
```

**それでも解決しない場合**
エラーメッセージをそのままコピーして、Claudeに貼り付けてください。
